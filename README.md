#  Bike Store Sales Dashboard

##  Sobre o Projeto

Este projeto consiste na criação de um **dashboard de análise de vendas** utilizando uma base de dados de uma loja de bicicletas.

O objetivo foi explorar os dados e gerar **insights de desempenho comercial**, além de trabalhar o **design e a experiência visual do dashboard**.

Diferente de outros projetos focados em modelagem de dados complexa, este trabalho teve como foco principal a **visualização de dados e a criatividade no design**, com a interface visual sendo planejada no **Figma**.

---

##  Objetivos do Projeto

- Analisar o **faturamento da loja**
- Identificar **top vendedores**
- Comparar **vendas por categoria**
- Avaliar **vendas por cargo**
- Analisar **evolução das vendas ao longo do tempo**

---

##  Principais Métricas do Dashboard

O dashboard apresenta:

-  **Faturamento total**
-  **Top 3 vendedores**
-  **Vendas por categoria**
-  **Total de vendas por cargo**
-  **Vendas por ano e mês**
-  **Tabela detalhada de vendas**

Também foram implementados **filtros interativos**, permitindo analisar os dados por:

- Categoria  
- Produto  
- Vendedor  
- Cargo  

---

##  Design do Dashboard

O layout do dashboard foi desenvolvido com foco em **experiência visual e clareza das informações**.

O **background e os elementos visuais** foram criados no **Figma**, trazendo uma identidade visual mais moderna para o dashboard.

Além disso, foi criado um **menu lateral interativo** para facilitar a navegação entre as páginas do painel.
